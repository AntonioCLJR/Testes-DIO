O que é HTML?

VOLTAR PARA O INICIO

HTML não é considerada uma linguagem de programação, mas sim uma linguagem de marcação, projetada para estruturar e organizar o conteúdo de uma página web. Através de uma coleção de elementos, o HTML permite que você delineie ou agrupe seções específicas do conteúdo, definindo como elas devem ser exibidas ou comportar-se na página.

Estrutura Básica do HTML

VOLTAR PARA O INICIO

A estrutura básica de um documento HTML é composta por uma série de elementos fundamentais que formam a espinha dorsal de qualquer página web. Todo documento HTML começa com a declaração "!DOCTYPE html", que informa ao navegador que tipo de documento ele está lidando

Em seguida, temos a tag "html", que encapsula todo o conteúdo da página. Dentro dela, encontramos duas seções principais: o head e o body. O head contém metadados sobre a página, como o título "title", links para folhas de estilo e scripts.

Já o body abriga todo o conteúdo visível ao usuário, como textos, imagens, links e outros elementos interativos.


Tags

VOLTAR PARA O INICIO

As tags são os blocos de construção do HTML. Elas servem para delimitar diferentes partes do conteúdo e definir como cada parte deve ser exibida no navegador.

Cada tag geralmente tem uma abertura e um fechamento. Por exemplo, a tag p é usada para criar um parágrafo de texto, enquanto a tag h1 define um título de maior importância.

Algumas tags, como img para imagens ou br para quebras de linha, são auto-fechadas, o que significa que não necessitam de uma tag de fechamento.


Atributos de Tags

VOLTAR PARA O INICIO

Os atributos das tags fornecem informações adicionais sobre os elementos HTML, permitindo uma customização mais precisa de como eles se comportam ou são exibidos.

Atributos são sempre incluídos na tag de abertura e consistem em um nome e um valor, separados por um sinal de igual, como em atributo="valor".

Por exemplo, a tag img usa o atributo src para definir o caminho da imagem e o atributo alt para fornecer um texto alternativo caso a imagem não carregue.

Outros atributos comuns incluem class, id, style, e href, cada um com funções específicas que ajudam a melhorar a acessibilidade, o estilo e a funcionalidade da página.


Textos

VOLTAR PARA O INICIO

O HTML oferece várias tags para formatar e estilizar textos, tornando-o uma ferramenta poderosa para a criação de conteúdo web.

A tag p é usada para definir parágrafos de texto, enquanto as tags de título h1 a h6 definem diferentes níveis de títulos, com h1 sendo o mais importante e h6 o menos.

Para adicionar ênfase ao texto, podemos usar strong para texto em negrito e em para itálico. Além disso, blockquote é usado para citações longas. Cada tags desempenha um papel na formatação textual e na apresentação do conteúdo.


Listas Ordenadas e Não Ordenadas

VOLTAR PARA O INICIO

Listas são uma forma eficaz de organizar informações em HTML. Existem dois tipos principais de listas: ordenadas e não ordenadas.

Listas ordenadas são criadas com a tag ol, e cada item da lista é colocado dentro de uma tag li, resultando em uma lista numerada automaticamente.

Por outro lado, listas não ordenadas são definidas com a tag ul, e seus itens também usam a tag li, mas são exibidos com marcadores ao invés de números.

Ambas as listas são úteis para estruturar dados de forma clara e organizada, tornando o conteúdo mais fácil de ser compreendido.

Veja um Exemplo de Lista Ordenada:

  1. Nome
  2. Número
  3. Matricula

Agora veja um Exemplo de Lista Desordenada:

  • Nome
  • Número
  • Matrícula

Links

VOLTAR PARA O INICIO

Links são um dos elementos mais fundamentais da web, permitindo a navegação entre diferentes páginas e recursos.

Em HTML, links são criados usando a tag a, que significa "anchor".

atributo principal dessa tag é href, que define o destino do link. Por exemplo, a href="https://www.exemplo.com"> cria um link clicável que leva o usuário ao endereço especificado.

Além do atributo href, a tag a pode incluir atributos como target, que define se o link deve abrir em uma nova aba, e title, que oferece informações adicionais quando o usuário passa o mouse sobre o link.
